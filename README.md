## 🚀 Quick Start

1. **Download** `TrafficNovaPro_v1.0.0_Setup.exe` from the [Releases page](https://github.com/multidigitaltools/TrafficNovaPro/releases)
2. **Run as Administrator** — .NET 10 runtime is bundled, no separate download
3. **Start Trial** or paste your license key
4. **Add proxies** — Proxies page → Import → Validate All
5. **Create campaign** — target URL, session count, proxy pool, resource blocking
6. **Run or Schedule** — immediate or cron-based

📖 [Full User Guide →](https://multidigitaltools.com/docs/trafficnova-pro/)
